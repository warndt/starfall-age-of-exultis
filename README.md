# starfall-age-of-exultis
RPG game for Google Edge

Character Creation — conversational, one question at a time, no menu dumps. Khadim vs Am'elon origin, 6 backgrounds, stat distribution, and a generated starting inventory with a d6 coin roll.

5-Stat System — STR, AGI, WIT, RES, CHA — each mapped to real world paths. Bonuses flow naturally from origin and background choices.

Dice Mechanics — all narrated in prose. Partial successes, fumbles, heroic moments — never a clinical number readout. Combat uses descriptive health states (bloodied → staggered → broken).

9 Destiny Paths — each gated by stats + story. You can't just grind to become a Miral; you have to earn Capan first and build CHA through play. The Keeper path is female-only and requires a contact, as your lore implies.

Srysarc Arc — 5-stage unlock tied to story milestones, not grinding. The device responds to emotional extremes before it gives utility — feels alive, not just a gadget.

Opening — drops the player into District Seven of Caedra's lower strata, pre-Star Fall, with the world humming with tension but not yet erupting.

To install: place the expanse-dungeon-master/ folder in the skills/ directory of your Google AI Edge Gallery fork, then load it via URL or local file import in the app's Skill Manager. It works as a pure text-only skill — no JS needed, fully offline.
